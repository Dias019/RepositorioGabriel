Agenda de contatos

Sobre o projeto:

Explicação

A página da Agenda de Contatos foi desenvolvida com o objetivo de praticar o uso de PHP e armazenar informações de contatos de forma eficiente.

Utilizando PHP como a linguagem principal, esta agenda permite o armazenamento e gerenciamento de dados dos usuários em um banco de dados,

enquanto o frontend é estruturado com HTML e CSS básicos.

Como surgiu: O projeto surgiu como uma oportunidade para praticar habilidades em PHP, proporcionando

uma visão prática de como criar

e gerenciar uma aplicação que armazena dados de usuários.

Com o intuito de aprimorar meus conhecimentos em PHP e explorar as funcionalidades do banco de dados,

desenvolvi a agenda para entender melhor o fluxo de dados entre o frontend e o backend.

O projeto ajudou a descobrir como implementar operações CRUD básicas (criar, ler, atualizar e deletar) e gerenciar dados de forma eficaz.

Sempre gostei muito da parte de Dados, então sempre quis desenvolver uma página WEB com a integração ao Banco de Dados.

Tive a oportunidade de usar o CRUD em projeto pela primeira vez nesse projeto da Agenda, por isso tem um significado tão grande para mim.

Foi o primeiro passo com a parte de backend, onde pude aprender muito e praticar ainda mais,

me ajudando a deixar meus projetos cada vez mais complexos e competos.

Apesar de uma dificuldade ou outra, consegui desenvolver o projeto de forma objetiva e calra para qualquer publico.

Funcionalidades

1. Adicionar Contato:

Formulário de Adição: Permite adicionar novos contatos ao banco de dados preenchendo um formulário com nome, telefone e observações. O processo é simples e direto, facilitando a inclusão de novos dados.

2. Editar Contato:

Formulário de Edição: Facilita a atualização de informações existentes. Após selecionar um contato, um formulário pré-preenchido permite modificar os dados e salvar as alterações.

3. Visualizar Contatos:

Lista de Contatos: Todos os contatos armazenados são exibidos em uma tabela, permitindo fácil visualização e acesso às informações. Cada contato pode ser visualizado em detalhes ao ser clicado.

4. Excluir Contato:

Opção de Exclusão: Contatos indesejados podem ser removidos do banco de dados com um simples clique.

Ferramentas Utilizadas

1. Linguagem de Programação:

PHP: Utilizado para criar a lógica do backend da agenda de contatos, gerenciar a interação com o banco de dados e implementar funcionalidades CRUD.

2. Banco de Dados:

MySQL: Utilizado para armazenar as informações dos contatos de forma estruturada e eficiente.

3. Frontend:

HTML/CSS: Utilizados para estruturar e estilizar a interface da agenda, garantindo uma apresentação clara e funcional.

4. Ferramentas de Desenvolvimento:

Editor de Código: Utilizado para escrever e editar o código PHP, HTML e CSS.

Ambiente de Desenvolvimento Local: Ferramentas como XAMPP ou WAMP são usadas para criar um ambiente de desenvolvimento local para testar e depurar a agenda antes da implantação.
