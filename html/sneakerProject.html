Sneaker Project

Explicação

Eu desenvolvi esse projeto para testar meus conhecimentos em desenvolvimento WEB.

Eu tinha curiosidade de saber como um

Carrinho de compras funcionava no JavaScript.

Então, criei a interface do projeto, e com alguns vídeos

no YouTube, acabei desenvolvendo esse carrinho de compras.

Por fim, juntando a interface com o Carrinho de compras,

foi desenvolvido esse projeto de loja online.

Funcionamento do Site

O site é uma loja de tênis online, onde o usuário entra,

procura o tênis que deseja e adiciona ao carrinho.

O usuário consegue adicionar quantos tênis quiser ao carrinho.

Sempre aparecerá uma mensagem dizendo que o tênis foi adicionado ao carrinho.

Após adicionar ao carrinho, o usuário irá abrir o carrinho de compras

e ver o valor total de seu pedido.

Em seguida, ele poderá escolher a forma de pagamento:

Seja no PIX ou Débito com 15% de desconto,

No cartão de crédito à vista, com 10% de desconto,

2x sem juros e sem desconto,

3x, 4x, 5x com um juros de 10%.

Sempre que o usuário escolher a forma de pagamento, o valor será atualizado,

aplicando o desconto ou o Juros, o número de parcelas e o novo valor total.

Hospedagem e Dificuldades

Como nunca tinha hospedado nenhum site sozinho ainda,

tive a ideia de me propor esse desafio, e ver se ia dar certo.

Primeiro de tudo, fui ver alguns vídeos de como era feita

essa hospedagem na web.

Eu já sabia de algumas plataformas que hospedavam sites gratuitamente,

e após ver alguns vídeos, fui realizar minha tentativa.

Realizei diversas tentativas, mas nenhuma me dava resultados,

pois o método que eu estava usando para subir meus arquivos no GitHub

estava errado.

Olhei mais alguns vídeos, e então comecei a subir meus arquivos via Git

diretamente no VS Code, e mesmo assim, não dava certo.

Após olhar mais alguns vídeos, acabei percebendo que faltava criar mais um arquivo

que faria com que o deploy do meu site funcionasse.

Criei esse arquivo, fiz a adição dos meus arquivos no GitHub através de um commit,

e quando atualizei o site, tinha dado certo.

O site que usei para hospedar gratuitamente foi o Netlify

Ferramentas

Como era apenas um teste dos meus conhecimentos,

optei por criar um site em vez de um app.

Então, para criar a interface (frontend), foram

utilizados o HTML e o CSS.

Para realizar a dinâmica do carrinho de compras,

eu utilizei o JavaScript.

Para criar o Modal do carrinho de compras, foi utilizada uma tag do HTML

chamada "dialog".

Para fazer o funcionamento dessa tag, é necessário criar uma lógica

no seu script do JavaScript.

Para criar a barra de formas de pagamento,

eu utilizei a tag do HTML "datalist", que nos permite

criar mais de uma opção para fornecermos ao usuário.

Para "inserir" os ícones em meu site, eu utilizei

o site Font Awesome, que nos permite escolher diversos

tipos de ícones, assim como eu escolhi os ícones das redes

sociais, como o Instagram, WhatsApp e GitHub.
